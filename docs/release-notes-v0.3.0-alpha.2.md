# SV-Gap v0.3.0-alpha.2

This redaction release replaces withheld provider model identifiers with stable
public configuration aliases throughout repository history, result provenance,
release assets, and container sources.

No evaluation outcome, RTL candidate, prompt, checker result, task contract, or
research claim changed. The alias mapping is intentionally one-way in public
artifacts.

The release retains the v0.3 adoption features:

- one-command `svgap demo`;
- bring-your-own-RTL documentation;
- multidimensional generation, diagnosis, and repair result profiles;
- a schema-validated public result registry; and
- open-source package, container, CI, and documentation paths.

The built-in backend remains a narrow research reference, not silicon signoff.
