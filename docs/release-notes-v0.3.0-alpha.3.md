# SV-Gap v0.3.0-alpha.3

This prerelease turns the compact research artifact into a model-neutral
collaboration surface for frontier-model researchers and chip-design AI
builders.

## What is new

- Connect any generator through a stdin/stdout command and separate generation
  from evaluation so generated RTL can be checked without model credentials.
- Run frozen generation, diagnosis, and repair workflows while preserving
  functional failure, structural failure, unknown, and tool-error outcomes.
- Create, validate, publication-scan, and deterministically bundle a result
  submission with public, attested-alias, or anonymous-case-study provenance.
- Publish accepted submissions as static, citable evidence profiles rather
  than a scalar leaderboard.
- Join a documented frontier RTL handoff study with clear contribution credit,
  response expectations, and claim boundaries.

## Interpretation and safety

Generated RTL is untrusted input. The documented two-stage workflow evaluates
it in a credential-free, network-disabled container. The built-in checker is a
narrow open reference backend, not silicon signoff. The real reset-release
perturbation instrumenter remains unavailable pending patent and employer
review.

See `docs/evaluate-your-model.md`, `docs/submitting-results.md`, and
`docs/collaboration-rfc.md` for the supported paths.
